באילו תקופות קורים אירועים מסוג סוים
ועוד

שם האתר + בעלים. בצורה כזאת ניתן לקבל מידע אילו אתרים עובדים בשיתוף פעולה עם ynet ולהסיק מסקנות בעת הצורך
